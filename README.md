# AI Code Error Fixer 

A high-performance FastAPI application deployed on Vercel that leverages the **Groq LPU Inference Engine** to diagnose and fix programming errors instantly.

##  Features
- **FastAPI Backend:** Lightweight and asynchronous Python framework.
- **Groq AI Integration:** Uses ultra-fast LLMs (like Llama 3) to analyze code errors.
- **Vercel Optimized:** Configured for Serverless deployment using Vercel Functions.
